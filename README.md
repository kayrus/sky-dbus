# sky-dbus
D-Bus example for Skype for Business on Linux (https://tel.red)

## Compile

### Ubuntu

```sh
$ apt update
$ apt install make qt5-default g++ -y
$ qmake
$ make
```

## Examples

### Join the meeting URL

```sh
$ ./sender jm https://meeting_link.com/corp/user
```
